## How to Use the Drawing Tool
- Use the Color Picker to choose between the three colors (Red, Blue, Yellow)
- Type in your desired brush size
- Drag your mouse (or finger) along the table to draw
- select the eraser to erase small details, reset to clear the whole board
- Happy Drawing!
